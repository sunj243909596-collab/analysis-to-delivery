ASN 单状态流转图（ASCII）

   ┌──────────┐
   │  创建 ASN │  (TMS 推单)
   └─────┬────┘
         │
         ▼
   ┌──────────┐
   │  10 新建  │  待收货
   └─────┬────┘
         │ 仓管员扫码开始收货
         ▼
   ┌──────────┐
   │  30 收货中│  至少 1 条明细已收
   └─────┬────┘
         │
         ├──→ 短收/拒收 → 推送质检员 ──→ 异常处理 ──→ 回到 30
         │                                            │
         │  (回流闭环：异常处理完毕回到收货中)         │
         │  ◄─────────────────────────────────────────┘
         │
         │ 全部明细收完 + 复核通过
         ▼
   ┌──────────┐
   │  40 收货完成│  复核员审核通过，触发 PIX 记账
   └─────┬────┘
         │
         │ (可驳回) → 回到 30 收货中
         │
         │ 全部完成
         ▼
   ┌──────────┐
   │  90 归档  │  (后续阶段)
   └──────────┘

异常分支（独立状态）：
   ┌──────────┐
   │  50 拒收  │  整单拒收（效期/质量不合格）
   └──────────┘

   ┌──────────┐
   │  60 取消  │  人工取消
   └──────────┘


LPN 货箱状态流转：

   ┌──────────┐
   │  10 未上架│  收货完成、货箱已建
   └─────┬────┘
         │ 仓管员扫码上架
         ▼
   ┌──────────┐
   │  30 已上架│  货箱已绑定货位
   └─────┬────┘
         │
         │ (后续：出库/移库时回到 10)
         │  ◄────────────┐
         └────────────────┘
            (回流闭环)


序列号状态流转（按 wmos-serial-tracking 规范）：

   00 → 10 → [30 / 40 / 10] → [M04 / 60] → 70 → 90

   00 初始 → 10 正常 → 30/40 中间状态 → M04/60 拣选/出库 → 70 复核 → 90 归档
