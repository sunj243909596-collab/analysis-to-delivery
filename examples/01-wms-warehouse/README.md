## 行业与技术栈

- **行业**：医药物流（GSP 强合规）
- **后端**：Java + Spring Boot + MyBatis-Plus + Oracle
- **前端**：Vue 3 + Element Plus
- **巴枪端**：uni-app + uView

## 关键合规要求

按 `config/compliance/gsp.md`：
- **00201** 药品追溯系统（严重）
- **05805** 计算机系统（严重）
- **05701** 全过程计算机系统（主要）
- **05901** 数据授权（主要）
- **04001-04003** 更改留痕（主要）
- **第八十五条** 效期锁定（一般）
- **第八十七条** 质量锁定（一般）

## 演示的业务场景

**入库收货管理** — 仓库到货后，仓管员用巴枪扫码收货，按 GSP 要求做：
1. 扫码 ASN 单号
2. 扫码实物（SKU + 批号 + 序列号 + 效期）
3. 校验批号、效期、质量状态
4. 上架到货位
5. 触发 PIX 事务，记账到 WMOS

涉及的 WMOS 原生表：`ASN`、`ASN_DETAIL`、`LPN`、`INVENTORY` 等。
