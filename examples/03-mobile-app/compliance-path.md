# 合规路径(移动 App — 会员积分管理)

> 项目:移动 App — 会员积分管理
> 路径配置:本文档被 `/setup-analysis-delivery` 阶段 0 引用
> 用途:明确本项目的合规要求与法规依据

## 一、合规等级

**等级:轻度(NONE + PIPL 自检)**

本项目为 B2C 移动 App,主市场中国大陆,**需遵循《个人信息保护法》(PIPL)** 自检要求。不涉及 GSP / 等保 / HIPAA / PCI-DSS。

**依据**:
- 收集手机号 → PIPL 个人信息
- 头像 / 昵称 → PIPL 个人信息
- 行为分析 → PIPL 个人信息
- 推送 → PIPL 个人信息
- 海外用户少 → GDPR / CCPA 暂不强制,但代码层面预留

## 二、《个人信息保护法》(PIPL)核心要求

### 2.1 收集规则

- ✅ **最小必要**:只收集业务必需的手机号 / 昵称 / 头像
- ✅ **明示同意**:首次启动展示隐私政策 + 用户协议,需用户主动勾选
- ✅ **单独同意**:敏感个人信息(位置 / 通讯录)需单独弹窗
- ✅ **儿童保护**:14 岁以下需监护人同意(本项目 v1 不允许儿童注册)

### 2.2 使用规则

- ✅ **目的限定**:只能用于积分业务,不可用于广告(除非用户同意)
- ✅ **最小权限**:App 端只申请必需权限(相机 / 存储 / 通知)
- ✅ **透明可查**:用户可查看自己的积分流水 / 个人信息

### 2.3 存储规则

- ✅ **境内存储**:中国大陆用户数据存境内 Firebase(需 Firebase 中国版 / 阿里云 / 腾讯云替代)
- ✅ **加密存储**:手机号 / 身份证等敏感字段加密
- ✅ **限期保存**:用户注销后 30 天内删除数据

### 2.4 用户权利

- ✅ **知情权**:隐私政策公开透明
- ✅ **决定权**:用户可关闭推送 / 退出登录 / 注销账号
- ✅ **查询权**:用户可导出个人数据(JSON 格式)
- ✅ **更正权**:用户可修改昵称 / 头像
- ✅ **删除权**:注销账号 = 删除数据
- ✅ **解释权**:用户对算法 / 自动决策有异议权

## 三、App Store / Google Play 上架合规

### 3.1 iOS App Store

- ✅ **App 隐私详情**:声明收集的数据类型(联系信息 / 标识符 / 使用数据)
- ✅ **追踪透明度**(ATT):iOS 14.5+ 需 IDFA 弹窗
- ✅ **Sign in with Apple**:第三方登录必须同时提供 Apple ID
- ✅ **权限申请说明**:每个权限需说明用途

### 3.2 Google Play

- ✅ **数据安全表单**:声明数据收集 / 存储 / 共享
- ✅ **权限政策**:敏感权限需说明
- ✅ **广告 ID**:广告 ID 使用需声明
- ✅ **家庭政策**:儿童应用需额外审核

## 四、Firestore 数据合规

| 字段 | PIPL 类别 | 存储方式 | 用户权利 |
|---|---|---|---|
| 手机号 | 敏感个人信息 | AES-256 加密 | 可见 / 修改 / 删除 |
| 头像 URL | 个人信息 | Storage HTTPS | 可见 / 修改 / 删除 |
| 昵称 | 个人信息 | 明文 | 可见 / 修改 |
| 积分余额 | 业务数据 | 明文 | 可见 |
| 积分流水 | 业务数据 | 明文 | 可见 |
| 行为日志 | 个人信息 | 脱敏(uid)| 可导出 |
| 推送 token | 个人信息 | 明文 | 可撤销 |

## 五、数据生命周期

| 数据 | 保留期 | 过期处理 |
|---|---|---|
| 用户基本档案 | 注销后 30 天 | 软删除 + 30 天后硬删除 |
| 积分余额 | 永久(业务需要) | 注销时清零 |
| 积分流水 | 5 年(财务合规)| 脱敏归档 |
| 行为日志 | 180 天 | 自动清理 |
| 推送 token | 失效即删 | FCM 自动 |
| 登录日志 | 90 天 | 自动清理 |

## 六、隐私政策模板(必备)

```markdown
# 隐私政策(示例,需法务审核)

## 一、我们收集什么
1. 注册信息:手机号 / 昵称 / 头像
2. 设备信息:设备型号 / OS 版本 / 推送 token
3. 使用信息:签到 / 兑换 / 浏览

## 二、用于什么
1. 提供积分服务(主要目的)
2. 推送通知(用户可关闭)
3. 改进产品(脱敏分析)

## 三、如何存储
1. 境内服务器(中国)
2. AES-256 加密
3. 90 天滚动删除日志

## 四、用户权利
1. 查看 / 修改个人信息
2. 导出个人信息(JSON)
3. 删除账号(30 天内彻底删除)
4. 关闭推送通知

## 五、联系方式
- 邮箱:privacy@example.com
- 地址:北京市朝阳区 XXX

## 六、生效日期
2026-06-22
```

## 九、合规路径选择

- ✅ **选择 PIPL** — 国内必选
- ✅ **选择 App Store / Google Play 政策** — 上架必选
- ⚠️ **预留 GDPR** — 海外用户自检
- ❌ **不选择 GSP** — 非医药
- ❌ **不选择等保** — v1 非政企
- ❌ **不选择 HIPAA** — 非医疗
- ❌ **不选择 PCI-DSS** — 无支付
