> 版本：v0.1
> 日期：2026-07-10
> 状态：✅ 已确认

## 一、测试范围

本测试用例覆盖移动 App 会员积分管理的全部功能,包括：
- 用户注册/登录（手机号/微信/Apple ID）
- 积分获取（签到/消费返/邀请/活动）
- 积分兑换（商品/优惠券）
- 积分明细与过期（FIFO 批次）
- 推送通知（FCM/APNs）
- 离线支持
- 店员核销端
- PIPL/App Store/Google Play 合规校验
- 性能与安全

测试范围以 `01-业务需求文档 BRD.md` §四 功能模块为准,涵盖 4.1-4.8 共 8 个核心模块 + 合规 + 性能 6 大维度。

## 二、测试策略

| 测试类型 | 范围 | 工具 |
|---|---|---|
| 单元测试 | Dart(Flutter)/ TypeScript(Cloud Functions) | flutter_test + jest |
| 集成测试 | App → Cloud Functions → Firestore emulator | firebase_emulator_suite |
| E2E 测试 | 真实设备 iOS/Android 全链路 | integration_test + Detox |
| 性能测试 | 签到响应 < 500ms + App 启动 < 2s | Firebase Performance + 手工 |
| 安全测试 | Firestore 安全规则 + 加密 + 权限 | emulator 规则测试 + 手工 |
